# 6.0 JPA Básico

## Decripción
Página creada para el curso de Desarrollo web. Un video mostrando al detalle el desarrollo se encuentra en la siguiente URL: 
<a href='https://youtu.be/UvpZsVVtywU'> Curso JPA y Hibernate desde CERO | Relaciones, CRUD y Spring Boot | Desarrollo web </a>.

Los temas que se explican son los siguientes:
- JPA
- H2
- CRUD y consultas básicas
- Integración con Thymeleaf
  
Puede acceder a una demo a través del siguiente enlace: <a href='https://six-jpa-basico.onrender.com'>JPA básico</a>.


## Ramas
Encontrará 4 ramas en el repositorio: main, final, inicial y video
- inicial: contiene los archivos para seguir el paso a paso del video
- final: versión con mejoras estéticas y algunas funcionales
  - Se mejoras visualmente la página
  - Se permite la busqueda en la barra de navegación
  - Se eliminan archivos de clases pasadas
  - Se redirecciona desde el home
  - Se arregla el modificar, en especifico la selección de la carrera
  - Se arregla bus al iniciar los datos
- Main: mismo contenido de la rama final e incluye adicional el dockerfile
- video: Mismo contenido que al acabar el video
